# 💝 ¡Bienvenida a tu DAM Organizer! 🚀

He preparado esta guía para que puedas instalar y empezar a usar tu propio organizador de tareas en un par de minutos. Solo tienes que seguir estos pasos:

## 1. Preparación 🛠️
Asegúrate de tener **Node.js** instalado en tu ordenador (es lo que hace que la app funcione). Si no lo tienes, descárgalo aquí: [https://nodejs.org/](https://nodejs.org/) (elige la versión **LTS**).

## 2. Borrar mis datos (Reset) 🧼
Como te he pasado mi carpeta, ahora mismo tiene mis tareas. Para empezar de cero con tu propio horario:
1.  Busca el archivo **`RESET_DATABASE.bat`** y ejecútalo.
2.  Escribe **`S`** y pulsa **Enter** cuando te pregunte.
3.  ¡Listo! La base de datos estará limpia y lista para tus asignaturas.

## 3. Instalar en tu Escritorio 🖥️
Para que no tengas que buscar esta carpeta cada vez que quieras estudiar:
1.  Busca el archivo **`INSTALAR_EN_ESCRITORIO.bat`** y ejecútalo.
2.  Ahora tendrás un icono llamado **DAM Organizer** en tu escritorio con un estilo Cyberpunk.

## 4. ¡A Estudiar! (Lanzamiento) 🚀
Cada vez que quieras usar la app:
1.  Usa el icono del escritorio o el archivo **`FULL_LAUNCH.bat`**.
2.  Se abrirán dos ventanas negras (no las cierres, solo minimízalas).
3.  Automáticamente se abrirá el navegador con tu panel de control.

## 5. ¿Quieres usarlo en tu Tablet Xiaomi? 📱
1.  Con el programa abierto en el PC, fíjate en la ventana negra del lanzador.
2.  Busca donde pone: `[INFO] Dirección para Tablet: http://192.168.1.XX:5173`.
3.  Escribe ese número en el Chrome de tu tablet.
4.  *Tip:* Dale a los 3 puntitos en Chrome y selecciona **"Instalar aplicación"** para tenerla a pantalla completa.
