# 💖 ¡Bienvenida a tu propio Organizador! 🚀

He preparado este espacio para que tengas tu propia lista de tareas y módulos, totalmente independiente y privada.

## 1. Cómo acceder desde tu Tablet 📲
Tu organizador funciona las 24 horas en el NAS. Para entrar:

1.  Asegúrate de tener **Tailscale** activo en tu tablet o móvil.
2.  Abre el navegador y escribe: `http://100.72.183.18:5174`
    *(Nota: Usamos el puerto **5174** y el sistema ya está configurado para ser 24/7).*

## 2. Instalar como App ✨
Para que sea más cómodo:
1.  En Safari (iPad) o Chrome (Android), pulsa en el menú y selecciona **"Añadir a la pantalla de inicio"**.
2.  Ahora tendrás un icono directo que se abre a pantalla completa.

## 3. Empezar de cero 🧼
He limpiado la base de datos para que puedas crear tus propios módulos (Asignaturas) y tareas. Solo tienes que darle al botón de **"+"** para empezar a añadir cosas.
